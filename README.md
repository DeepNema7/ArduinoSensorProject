# 🌐 Arduino Sensor Project (IoT Based Monitoring System)

## 📌 Overview
This project is an IoT-based sensor monitoring system built using Arduino/NodeMCU.  
It integrates multiple sensors like Ultrasonic, IR, and CO₂ to collect real-time environmental data and upload it to the cloud using ThingSpeak.

The system helps in monitoring distance, obstacle detection, and air quality efficiently.

---

## 🚀 Features
- 📏 Distance measurement using Ultrasonic Sensor
- 🚧 Object detection using IR Sensor
- 🌫️ CO₂ / Air quality monitoring
- ☁️ Real-time data upload to ThingSpeak
- 📊 Data visualization on cloud dashboard
- 🔌 Low-cost and scalable IoT solution

---

## 🛠️ Components Used
- NodeMCU (ESP8266) / Arduino
- Ultrasonic Sensor (HC-SR04)
- IR Sensor
- CO₂ Sensor (MQ series)
- Jumper Wires
- Breadboard
- Power Supply

---

## ⚙️ Working Principle
1. Sensors collect real-time data:
   - Ultrasonic → Distance
   - IR → Obstacle detection
   - CO₂ → Air quality level  

2. NodeMCU processes the data.

3. Data is sent to ThingSpeak via WiFi.

4. Users can monitor data remotely using charts.
